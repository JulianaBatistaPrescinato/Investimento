# Investimento
Planilha para simulação de investimentos

Simulador de Investimentos
Este projeto foi desenvolvido como parte de um desafio prático da DIO. 
O objetivo é aplicar conceitos de Excel desenvolvidos em aula para criar uma ferramenta de simulação que auxilie investidores.

Descrição do Projeto
O simulador automatiza cálculos financeiros, permitindo que o usuário visualize diferentes cenários de investimento. 
A planilha utiliza parâmetros como aporte mensal, taxa de rendimento esperada e tempo de investimento para calcular o montante final e a renda passiva gerada.

Principais Funcionalidades:
Cálculo baseado em salário e sugestão de aporte (ex: 30%).

Cálculo de Patrimônio: Projeção de acúmulo de capital para períodos de 2, 5, 10, 20 e 30 anos.

Estimativa de Dividendos: Visualização de quanto o investidor receberá mensalmente em cada cenário.

Alocação por Perfil: Sugestões de distribuição de carteira entre Papel, Tijolo, Híbridos, FOFs, Desenvolvimento e Hotelaria, segmentadas por perfis (Conservador, Moderado e Agressivo).

Tecnologias Utilizadas
Microsoft Excel: Fórmulas financeiras, organização de dados e modelagem de cenários.

Markdown: Para documentação do projeto no GitHub.

Estrutura da Planilha
A ferramenta é composta por:

Aba de Investimento: Onde o usuário insere o salário, valor do aporte mensal (ex: R$ 200,00), o tempo e a taxa de rendimento.

Base de Dados: Contém a inteligência de alocação por tipo de fundo imobiliário (percentuais) de acordo com a tolerância ao risco.

Cenários de Longo Prazo: Projeções automáticas que mostram o resultado esperado.

Como usar
Faça o download do arquivo Investimento.xlsx.

Abra no Microsoft Excel ou Google Sheets.

Na seção "CONFIGURAÇÕES", insira seu salário para ver a sugestão de investimento.

Na seção "INVESTIMENTO MENSAL", altere os campos de "Quanto investir", "Anos" e "Taxa" para simular sua realidade.

Analise os resultados na tabela de "Cenários".
